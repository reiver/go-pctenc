# go-pctenc

Package **pctenc** provides tools for working with **percent-encoding** (also known as **url-encoding**), for the Go programming language.

## Documention

Online documentation, which includes examples, can be found at: http://godoc.org/github.com/reiver/go-pctenc

[![GoDoc](https://godoc.org/github.com/reiver/go-pctenc?status.svg)](https://godoc.org/github.com/reiver/go-pctenc)

## Import

To import package **pctenc** use `import` code like the following:
```
import "github.com/reiver/go-pctenc"
```

## Installation

To install package **pctenc** do the following:
```
GOPROXY=direct go get github.com/reiver/go-pctenc
```
